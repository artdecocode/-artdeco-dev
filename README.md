# @artdeco/dev

`@artdeco/dev` is a new Node.js npm package.

Development version of the artdeco website.

## `artdecoDev():void`

Call this function to get a result you want.

```js
const artdecoDev = require('@artdeco/dev')

artdecoDev()
```
